# Fake News Headline Generator

A Python project that generates random fake news headlines using modular lists of subjects, actions, joiners, and places.

## Features
- Random headline generation
- Multiple categories
- Save headlines to text file
- Timestamp support
- User interaction loop

## Technologies Used
- Python
- random
- datetime
- file handling

## Sample Output
16-04-2026|14:25 BREAKING NEWS : Scientist launched mysterious object near Kolkata
