# Fake News Headline Generator

This is my first self-made Python project that generates **random fake news headlines** using modular lists of subjects, actions, joiners, and places.

The program randomly combines words from different categories to create unique headlines with **real-time date and time stamps**.

It also gives the user an option to **save the generated headlines into a text file**.

## Features
- Random headline generation
- Multiple categories
- Save headlines to text file
- Timestamp support
- User interaction loop

## Technologies Used
- Python
- random
- datetime
- file handling

## Sample Output
16-04-2026|14:25 BREAKING NEWS : Scientist launched mysterious object near Kolkata
